# REDIS

> Remote Dictionary Server, "키-값" 구조의 비정형 데이터를 저장하고 관리하기 위한 오픈 소스 기반의 비관계형 데이터베이스 관리 시스템이다. (NoSQL)

## 특징

1. **Remote 에 위치한다.**
2. **프로세스로 존재한다.**
3. **In-Memory - 메모리 기반이다.**
4. **“키-값” 구조 데이터 관리 시스템**
   * 비 관계형이며, 키-값 구조이기 때문에 별도 쿼리 없이도 데이터를 간단히 가져올 수 있다.
